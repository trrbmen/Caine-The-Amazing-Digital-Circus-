# 🔴 CAINE SYSTEM

A buggy artificial intelligence interface built for experimental interaction.

---

## What is CAINE?

CAINE is an unstable AI-like system designed to simulate conversation.

It is not fully functional.  
It behaves unpredictably.

Sometimes it responds correctly.  
Sometimes it does not.

Sometimes it talks on its own.

---

## Features

- Command-based interaction  
- Delayed responses (2–4 seconds)  
- Typing effect  
- Glitch visual effects  
- Self-generated messages (idle mode)  

---

## Commands

- /help → list commands  
- /joke → random response  
- /roast → sarcastic reply  
- /story → origin text  
- /bug → unstable output  
- /exit → system refusal
